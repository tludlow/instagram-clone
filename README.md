
Instagram clone
======================
This is one part of a series of frontend application clones I am making to help learn full stack web development in more detail
[View live site here](https://instagram-clone-git-master.tludlow.vercel.app/)

## Features
Uses a modern frontend tech stack (Next.JS, Typescript, Golang backend)
- [x] Navbar
	* Logo
	* Search functionality with icons, saved previous search and easily deleting the input
	* **TODO**: dropdown for search results
	* Navigation icons and the appropriate drop downs related to them
- [x] Stories
	* Carousel of stories 

More to come...



## Installation
If you wish to run the code, clone/fork the repository and install the node dependencies using either of the following commands from the root directory 
Yarn:
`cd frontend && yarn dev`

Node.JS
`cd frontend && node run dev`

Once this has been completed you should be able to see the running development version of the project in your browser at: [localhost:3000](http://localhost:3000)
